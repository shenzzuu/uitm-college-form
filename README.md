# 🏫 UiTM College Accommodation Application System

The **UiTM College Accommodation Application System** is a web-based platform developed to digitalize the student hostel application process.

This system allows students to apply for on-campus accommodation online, replacing traditional manual paper forms and improving efficiency for both students and college management.

Developed as an academic project for the Bachelor of Information Technology (Hons.) at Universiti Teknologi MARA (UiTM).

## 🎯 Project Objectives

- Replace manual hostel application forms with a digital system
- Improve efficiency in student data collection
- Reduce paperwork and administrative workload
- Provide structured data management for college administration

## 👩‍🎓 User Roles

### 🧑‍🎓 Student

- Fill in accommodation application form
- Submit application digitally
- Track application status

## 🚀 Key Features

- Digital application form with validation
- Responsive web interface
